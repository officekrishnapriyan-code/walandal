# WAL & AL - Premium Engineering Studio Website

## Overview
A completely reconstructed, high-end website for WAL & AL featuring a Deep Tech aesthetic with glassmorphism, Bento Grid layouts, and premium interactions.

## Design System

### Color Palette
- **Deep Slate**: `#0a0e1a` (Background)
- **Teal Accent**: `#0fb9b1` (Primary)
- **Sage/Earth**: `#7fb069` (Secondary)
- **Slate Grays**: `#475569`, `#64748b` (Muted elements)

### Typography
- **Primary Font**: Inter (Sans-serif) - Body text, headings
- **Technical Font**: JetBrains Mono - Labels, tags, technical specs

### Key Features

#### 1. Cinematic Hero Section
- Bold, centered headline with gradient text effect
- Animated teal-tinted radial glow
- Subtle grid pattern background
- Real-time stats with monospace font

#### 2. Bento Grid Layout
- Responsive grid system for Systems section
- Variable card sizes (large, medium)
- Technical specs with arrow bullets
- Glassmorphism effects on all cards

#### 3. Premium UI Elements
- **Glassmorphism**: `backdrop-filter: blur(24px)` on navigation and cards
- **Hover Effects**: Subtle glow, lift, and border color transitions
- **Smooth Animations**: Scroll-reveal with stagger effect
- **Custom SVG Icons**: Technical schematic style for domains

#### 4. Domains Section
- 4-column grid showcasing Water, Air, Land, All
- Custom SVG icons with technical aesthetic
- Monospace tags for categorization

#### 5. Responsive Design
- Mobile-first approach
- Sleek slide-in navigation menu
- Optimized touch targets (44px minimum)
- Graceful degradation on smaller screens
